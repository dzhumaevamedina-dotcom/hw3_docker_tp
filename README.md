# hw3_docker_tp

Проект состоит из двух частей:
- `generator` - генератор CSV-файла;
- `reporter` - скрипт для создания HTML-отчета;
- `data` - папка для данных,которые создаются через Docker;
- `local_data` - папка для данных,созданных локально;
- `run.sh` - основной bash-скрипт для запуска команд.

# Основные команды
Перед запуском можно дать файлу run.sh право на выполнение:
chmod +x run.sh

Сборка и запуск генератора:

./run.sh build_generator
./run.sh run_generator

Сборка и запуск аналитика:

./run.sh build_reporter
./run.sh run_reporter

После запуска в папке data появляются файлы data.csv и report.html.
